NFL Data Visualizer [2023]

Created a real-time data visualizer which utilized the 2022 Big Data Bowl which contains Next Gen Stats for all 2018-2020 Special Team plays. The data was parsed to optimize performance and keep the visualization as real-time as possible while not sacrificing performance. I've made this project in C++/C# and also in HTML/JavaScript using Observable to embed it onto my website!
